# BMC Menu Banner

Reusable Balcony Music Club menu banner component for HTML source embeds.

## Files

- `index.html` — standalone preview page for GitHub Pages.
- `site-nav.css` — banner styling copied from the working BMC / Upcoming Shows header pattern.
- `site-nav.js` — mobile menu toggle behavior.
- `banner.html` — header markup only.
- `embed.html` — copy/paste embed block for Bandzoogle / HTML source embeds.

## Expected GitHub Pages URL

After this repo is created as `floydclaptonblues/BMC-Menu-Banner` and GitHub Pages is enabled from the main branch root:

```text
https://floydclaptonblues.github.io/BMC-Menu-Banner/
```

## Embed block

Use the contents of `embed.html` in an HTML source/embed block.

To mark a page as active, add `class="active" aria-current="page"` to the appropriate nav link and remove those attributes from the others.
