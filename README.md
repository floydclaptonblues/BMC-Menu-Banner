# BMC Menu Banner

Reusable Balcony Music Club menu banner component for HTML source embeds.

## Files

- `index.html` — standalone GitHub Pages preview.
- `site-nav.css` — menu banner styling from the working BMC/Upcoming Shows header pattern.
- `site-nav.js` — mobile menu toggle behavior.
- `banner.html` — header markup only.
- `embed.html` — copy/paste embed block for Bandzoogle or other HTML source embeds.

## GitHub Pages URL

```text
https://floydclaptonblues.github.io/BMC-Menu-Banner/
```

## How to embed

Copy the full contents of `embed.html` into an HTML source/embed block.

To mark the current page, add:

```html
class="active" aria-current="page"
```

to the correct navigation link.
